# X-Drone 前端技术文档

## 1. 技术栈概述

### 1.1 核心技术

本项目采用纯原生 JavaScript 开发，无框架依赖，具有以下技术特点：

- **JavaScript**: 纯原生 ES5/ES6+，无框架依赖
- **CSS3**: CSS3 + CSS Variables，支持响应式布局
- **HTML5**: HTML5语义化标签，现代页面结构
- **架构模式**: 模块化单文件架构，功能分离

### 1.2 技术选型理由

| 技术 | 选型理由 |
|------|---------|
| 纯 JavaScript | 无构建依赖、轻量级、快速加载、易于维护 |
| CSS Variables | 便于主题管理、响应式适配、统一设计规范 |
| 模块化架构 | 功能分离、代码可维护性强、便于协作开发 |
| localStorage | 客户端数据持久化、无需额外存储方案 |

### 1.3 第三方依赖

- **Font Awesome 6.0.0-beta3**: 图标库（CDN引入）
- **无其他依赖**: 保持轻量化

### 3.2 聊天功能实现

#### 3.2.1 打字机效果

```javascript
function typeMessage(element, text, speed = 30) {
    if (currentTypingController) {
        currentTypingController.cancel();
    }
    
    let isCancelled = false;
    let index = 0;
    
    const controller = {
        cancel: function() {
            isCancelled = true;
        }
    };
    
    currentTypingController = controller;
    
    return new Promise((resolve) => {
        element.innerHTML = '<span class="typing-cursor"></span>';
        const cursor = element.querySelector('.typing-cursor');
        
        function typeNext() {
            if (isCancelled) {
                element.innerHTML = escapeHtml(text);
                resolve();
                return;
            }
            
            if (index < text.length) {
                const textNode = document.createTextNode(text[index]);
                element.insertBefore(textNode, cursor);
                index++;
                chatBox.scrollTop = chatBox.scrollHeight;
                setTimeout(typeNext, speed);
            } else {
                cursor.remove();
                resolve();
            }
        }
        
        setTimeout(typeNext, speed);
    });
}
```

**关键特性**:
- 支持取消当前打字效果
- 光标闪烁动画 (`typing-cursor`)
- 自动滚动到最新消息
- 防XSS处理

```javascript
async function sendMessage() {
    const msg = userInput.value.trim();
    if (!msg) return;
    
    cancelTyping();  // 取消当前打字效果
    
    // 用户消息立即显示
    addMessage('user', msg, true);
    userInput.value = '';
    
    showTypingIndicator();  // 显示AI思考动画
    
    const res = await fetch(`${API_BASE}/api/chat`, {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({
            user_id: currentUserId || 'guest',
            session_id: sessionId,
            message: msg
        })
    });
    
    removeTypingIndicator();
    
    const data = await res.json();
    addMessage('ai', data.response, false, true);  // 使用打字机效果
}
```

### 3.4 后台管理实现

#### 3.4.1 权限控制

```javascript
async function fetchWithAdminToken(url, options = {}) {
    if (!currentToken) {
        showToast('请先登录再访问后台', 'warning');
        showLoginModal();
        throw new Error('NotLoggedIn');
    }
    
    const headers = {
        ...options.headers,
        'Authorization': `Bearer ${currentToken}`
    };
    
    const response = await fetch(url, { ...options, headers });
    
    if (response.status === 401 || response.status === 403) {
        showToast('用户权限不够无法访问', 'error');
        goToChat();
        throw new Error('Forbidden');
    }
    
    return response;
}
```

### 5.2 防抖与节流

```javascript
// 防抖：延迟执行，重复调用会重置计时
function debounce(fn, delay = 300) {
    let timer = null;
    return function(...args) {
        if (timer) clearTimeout(timer);
        timer = setTimeout(() => {
            fn.apply(this, args);
            timer = null;
        }, delay);
    };
}

// 节流：固定频率执行，不会重复触发
function throttle(fn, delay = 300) {
    let last = 0;
    let timer = null;
    return function(...args) {
        const now = Date.now();
        const remaining = delay - (now - last);
        
        if (remaining <= 0) {
            if (timer) { clearTimeout(timer); timer = null; }
            last = now;
            fn.apply(this, args);
        } else if (!timer) {
            timer = setTimeout(() => {
                last = Date.now();
                timer = null;
                fn.apply(this, args);
            }, remaining);
        }
    };
}
```

**使用场景**:
- 用户名实时校验 → 防抖
- 滚动事件监听 → 节流
- 搜索输入框 → 防抖

### 5.3 按钮加载状态

```javascript
function setButtonLoading(btn, loading, loadingText = '处理中...') {
    if (!btn) return;
    
    if (loading) {
        if (btn.dataset.originalText === undefined) {
            btn.dataset.originalText = btn.innerHTML;
        }
        btn.disabled = true;
        btn.innerHTML = `<i class="fas fa-spinner fa-spin"></i> ${loadingText}`;
        btn.style.opacity = '0.7';
        btn.style.cursor = 'not-allowed';
    } else {
        btn.disabled = false;
        btn.innerHTML = btn.dataset.originalText;
        btn.style.opacity = '';
        btn.style.cursor = '';
        delete btn.dataset.originalText;
    }
}

// 使用示例
setButtonLoading(loginBtn, true, '登录中...');
// API调用后
setButtonLoading(loginBtn, false);
```
